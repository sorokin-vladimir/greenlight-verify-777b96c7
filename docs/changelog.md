# Changelog

Все значимые изменения проекта документируются в этом файле.

## [v1] — 2026-04-22

Первый релиз лендинга **Verifio** — KYC-сервиса с акцентным зелёным цветом.

### Дизайн-система
- Палитра **Emerald Prestige**: глубокий изумрудный (`--primary`) + яркий мятный акцент (`--accent`), все цвета в HSL.
- Типографика: **Sora** (заголовки) + **Manrope** (основной текст).
- Кастомные градиенты: `--gradient-emerald`, `--gradient-mint`.
- Утилитарные анимации: `animate-float`, `animate-fade-up`.
- Семантические токены и расширенный `tailwind.config.ts` (цвета, шрифты, радиусы).

### Компоненты
- Расширен `Button` новыми вариантами: `accent` (яркий зелёный) и `hero` (градиент).
- Создан модульный набор секций в `src/components/landing/`:
  - `Header.tsx` — навигация лендинга.
  - `Hero.tsx` — первый экран с визуальным мок-демо процесса верификации.
  - `BentoFeatures.tsx` — возможности платформы в bento-сетке.
  - `HowItWorks.tsx` — пошаговое описание работы сервиса.
  - `Coverage.tsx` — поддерживаемые документы и страны.
  - `Pricing.tsx` — тарифные планы.
  - `FAQ.tsx` — часто задаваемые вопросы.
  - `CTA.tsx` — финальный призыв к действию.
  - `Footer.tsx` — подвал.
- Вспомогательный `NavLink.tsx`.

### Структура страницы
- Bento-сетка как основа лендинга.
- Порядок секций на `Index.tsx`: Hero → Pricing → HowItWorks → Coverage → BentoFeatures → FAQ → CTA → Footer.
  (Блоки «Цены» и «Возможности» поменяны местами по запросу.)

### SEO и метаданные
- Локализованный `index.html`: `lang="ru"`, заголовок, описание, Open Graph теги.
- Адаптивный viewport, семантическая разметка.

### Документация
- Добавлена папка `docs/` с файлом `changelog.md`.
